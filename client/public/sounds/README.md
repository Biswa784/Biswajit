# Sound Files

Add your MP3 sound files to this directory. Required files:

- `open.mp3` - When opening windows
- `close.mp3` - When closing windows
- `click.mp3` - Button clicks
- `minimize.mp3` - When minimizing windows
- `maximize.mp3` - When maximizing windows
- `drag.mp3` - When dragging windows
- `error.mp3` - Error sound
- `startup.mp3` - On app startup (plays after first click)
- `shutdown.mp3` - On app shutdown
- `folder.mp3` - When opening folder
- `file.mp3` - When opening file

## How to Get Sounds:

### Option 1: Use Online Sound Libraries (Free)
- [Freesound.org](https://freesound.org/) - Creative Commons sounds
- [Zapsplat](https://www.zapsplat.com/) - Free sound effects
- [Mixkit](https://mixkit.co/free-sound-effects/button/) - Free UI sounds

### Option 2: Record Your Own
- Use Audacity (free) or your OS recording tool
- Export as MP3

### Option 3: Use Web Sounds (NOT RECOMMENDED)
- The app is configured for local files to avoid:
  - CORS issues
  - Internet dependency
  - Loading delays

## Tips:
- Sounds should be short (0.5-2 seconds)
- Keep file size small (50-200 KB each)
- Use consistent volume levels
