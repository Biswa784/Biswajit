🚀 HOW TO DEPLOY TO VERCEL (3 Easy Steps)

OPTION 1: Automatic Deployment (Recommended for Windows)
────────────────────────────────────────────────────────
1. Double-click "deploy.bat" in the project folder
2. Follow the prompts (it will open browser for login)
3. Done! Your portfolio is live in minutes

OPTION 2: Using Vercel CLI
──────────────────────────
1. Open terminal/PowerShell
2. Navigate to project: cd portfolioBoot
3. Run: vercel --prod
4. Follow the login prompts
5. Deploy complete!

OPTION 3: Using Vercel Dashboard (Most Control)
───────────────────────────────────────────────
1. Go to https://vercel.com/dashboard
2. Click "Add New" → "Project"
3. Select "Import Git Repository"
4. Choose your GitHub repo
5. Click "Deploy"

🆘 TROUBLESHOOTING

Build fails?
→ Check client/package.json for all dependencies
→ Run: npm install in client folder
→ Clear node_modules: rm -r client/node_modules && npm install

Resume download not working?
→ Check file exists: client/public/resume.txt
→ Verify file permissions
→ Check browser console for errors

Projects not showing?
→ Backend server must be running
→ Check VITE_API_URL environment variable
→ Verify MongoDB connection

Email link not working?
→ Check mail client is configured on your system
→ Email address: biswajitpanda871@gmail.com (in code)
